- Desarrollo -

En esta práctica aprendimos a usar las herramientas básicas de SOLIDWORKS, que es una plataforma para diseñar en 3D. Los pasos para comenzar a diseñar son sencillos, primero se selecciona un plano (Alzado, Planta o Vista Lateral), posteriormente, se crea un Sketch y puedes comenzar tu diseño empleando las distintas herramientas de trazo que la plataforma ofrece. Por último se extruye el trazo y en caso de querer una pieza de mayor complejidad se tendrán que realizar distintos trazos o cortes.
